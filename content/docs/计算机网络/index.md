# 计算机网络层级

1. 应用层
2. 传输层
3. 网络层（数据平面/控制平面）
4. 数据链路层
5. 物理层

## 原理

提供的服务和实现的功能

功能：每一层所具备的能力，每层的功能的实现是通过调用下一层的服务

服务：功能的体现，指的是功能向上一层提供的接口

# 应用层

# 传输层

# 网络层
